cd - --возвращает в предыдущую директорию, в которой находился
cd ~ --возвращает в домашюю директорию
Ctrl + L --очистка экрана(тоже самое, что clear), но он не удаляет историю, поэтому можно листать вверх
reset -- clear на стероидах. Очищает полностью
Ctrl + Z --сворачивание файла, открытого в терминале(с помощью vim, на пример) fg --вернет в файл
htop --утилита для мониторинга ресурсов
sudo !! --выполняет последнюю введенную команду(к примеру apt update)с припиской sudo
Ctrl + R --вызывает поиск уже когда введенных команд(нужно начать писать часть команды, которую помнишь.
           Менять команды можно еще раз нажимая Ctrl + R)
history --вызывает историю введенных команд с номерными индексами напротив(Выполнить команду можно так: !номер_команды)
F11 --fullscreen terminal
Ctrl + Shift + '+' --increase font
Ctrl + Shift + '-' --decrease font
Ctrl + U --удаляет вводимую команду(удобно, если команда длинная)
tail -f path/to/file --логирует что происходит с файлом в реальном времени
